## Friend finder is an friend matching website
- Takes in input from questions and return matching applicants
- The questions are light hearted
- The user photo name and score is saved on the backend with node
- The site is hosted on heroku. 
## Friend finder have a link to the current api list
- users can see everyone who answered questions on the site 
- all info is saved in a json format
### Friend finder uses 2 node modules 
- express
- path

[Visit Friend finder](https://friend-finder-ron.herokuapp.com/)
